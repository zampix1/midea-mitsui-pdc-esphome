# Home Assistant

This repository includes optional Home Assistant files for migration from an older Raspberry Pi HTTP exporter.

```text
home-assistant/packages/pdc_mitsui.yaml
home-assistant/dashboards/pdc_mitsui_dashboard.yaml
```

Review entity IDs before copying these files into an existing Home Assistant installation. They assume the ESPHome entity names generated by the package and may need local adjustment.

The ESPHome package itself is the primary deliverable. Home Assistant package and dashboard files are examples, not a HACS integration.
